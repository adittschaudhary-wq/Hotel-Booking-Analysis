# Hotel-Booking-Analysis
# Hotel Booking Cancellation Analysis

## Overview
This project analyzes hotel booking data to identify factors affecting booking cancellations.

## Dataset
Hotel Booking Demand Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Visualization
- Business Insights

## Key Insights
- City hotels had higher cancellation rates.
- Repeated guests cancelled less frequently.
- Bookings made far in advance had higher cancellation rates.

## How to Run

pip install -r requirements.txt

Open the notebook using Jupyter Notebook.
